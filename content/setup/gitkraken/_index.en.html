To install GitKraken download the correct binary for your system.

Then follow the installation instructions provided on their website.

In the installation instructions you’ll see that you do not actually need to install Git separately to use GitKraken as it ships it’s own Git framework, NodeGit, which is primarily developed and maintained by members of the GitKraken development team. The reason we asked you to install git is that if you wanted to use git in another setting (through the command line, or integrations with VS Code or Rstudio) you would need git installed on your system.
