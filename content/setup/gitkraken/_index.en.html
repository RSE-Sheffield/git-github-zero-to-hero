Install GtiKraken

To install GitKraken download the correct binary for your system.

Then follow the installation instructions provided on their website.

{{% notice note %}} In the installation instructions you’ll see that you do not actually need to install Git separately to use GitKraken as it ships it’s own Git framework, NodeGit, which is primarily developed and maintained by members of the GitKraken development team. The reason we asked you to install git is that if you wanted to use git in another setting (through the command line, or integrations with VS Code or Rstudio) you would need git installed on your system. The principles of the workflow we’ll be showing you remain the same. {{% /notice %}}

Integrating GitKraken with GitHub

In order to effectively work with GitHub repositories through GitKraken, you will need to enable the integration and authenticate with GitHub. You can then use GitKraken to define or generate an SSH key that authenticates git operations on GitHub through GitKraken. Follow the instructions on the GitKraken site to perform both steps.
